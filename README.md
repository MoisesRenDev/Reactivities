Mini Red Social Project with .Net Core and React-TS
Backend:
.Net Core 7
Sql Server Database
Made with the Clean Architecture.
MediatR for CQRS.
Signal R
JWT Token
Cloudinary for save images
Dockerizing App
Authentication with identity framework

Client:
React
Typescript
Mobx
Sematic UI CSS
Axios
